Book Finder App

React / API / PDF Viewer
General info

A React-based web app that allows users to search for books and access their details, including PDF versions when available.

Integrated with the Google Books API, users can search by title or author and view detailed information about each book, including previews or downloadable PDFs.

Results

Smooth user experience with integrated PDF viewing.

The project is built with a modern React architecture, ensuring performance, accessibility, and responsive design across devices.
