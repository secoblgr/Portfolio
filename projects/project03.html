Real-Time Web Chat

ASP.NET Core / SignalR / Bootstrap
General info

A real-time web chat application built with ASP.NET Core and SignalR, enabling instant communication between users.

The frontend is designed with Bootstrap for a clean and modern interface. Users can join the chat, send messages instantly, and see other active participants in real time.

Results

Fast and stable real-time communication.

Using SignalR ensures low-latency, bidirectional data transmission between clients and the server. The ASP.NET Core backend provides scalability and strong performance.
