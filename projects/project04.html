Real-Time Chat Mobile App

Flutter / SignalR / ASP.NET Core API
General info

A cross-platform mobile chat app developed with Flutter, using SignalR and ASP.NET Core API for real-time communication.

The app supports Android and iOS devices. It includes features like live message updates, online user tracking, and message history, all synchronized in real time via SignalR.

Results

Cross-platform, modern, and fast chat experience.

SignalR provides seamless synchronization between users, while the ASP.NET Core API ensures secure data communication. The Flutter UI is clean, responsive, and user-friendly.
